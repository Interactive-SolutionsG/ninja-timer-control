1. General Information

Ninja Timer Control is a task scheduling and management application built with maximum respect for your privacy. This policy explains what information we collect or what we DO NOT collect and how we protect your privacy.

Developer: Interactive Solutions

Contact: interactive.sup.solutions@outlook.com

2. Privacy Commitment

Core Principle:

Ninja Timer Control DOES NOT collect, DOES NOT store, DOES NOT transmit and DOES NOT share any personal data.

3. Data We DO NOT Collect

This application DOES NOT collect any of the following:

Personal Data:

  • Name, surname, address
  • Email address
  • Phone number
  • Date of birth
  • Geographic location
  • IP address

Usage Data:

  • Application usage statistics
  • Telemetry or behavioral analytics
  • Usage times
  • Frequency of use

Technical Data:

  • Device information
  • Operating system details
  • Hardware specifications
  • Unique device identifiers
  • Error or diagnostic logs

Note on device identifiers:
Microsoft Store uses device identifiers to manage licensing and sync subscriptions across your devices. However, we as developers DO NOT have access to these identifiers. We only receive a "license valid" or "license invalid" response.

Task Data:

  • Your scheduled tasks
  • Execution times
  • Task names

Everything remains on your local machine.

4. Data Storage

Local-Only Data

All information you configure in Ninja Timer Control is stored exclusively on your local device in: %APPDATA%\

Files created locally include:

  • Task configurations
  • User preferences
  • Selected language
  • Scheduled tasks

These files:

  • Exist only on your computer
  • Are NOT synced to any server
  • Are NOT accessible to the developer
  • Can be deleted at any time

5. Communications with External Servers

No Required Internet Connection

Ninja Timer Control DOES NOT require an internet connection to function and DOES NOT communicate with any external server.

Exceptions:

1. Microsoft Store:

  • When downloading or updating from the Microsoft Store
  • To validate your subscription or trial period
  • Managed by Microsoft

2. Updates (Optional):

  • Only if you manually check for updates
  • NO personal data sent, only current app version

NO:

  • Third-party analytics services
  • Advertising networks
  • Tracking services
  • Cookies

    • Payment information

10. Data Security

Local Protection

Since all your data is stored locally:

  • You are responsible for the physical security of your device
  • Windows manages file access permissions
  • We recommend:
    • Use a Windows account password
    • Keep Windows updated
    • Use reputable antivirus software

We:

  • Do NOT store data on servers
  • Have NO risk of server-side data breach
  • Do NOT collect anything remotely
