2. Compromiso de Privacidad

Principio Fundamental:

Ninja Timer Control NO recopila, NO almacena, NO transmite y NO comparte ningun dato personal.

3. Datos que NO Recopilamos

Esta aplicacion NO recopila ninguno de los siguientes datos:

Datos Personales:

  • Nombre, apellidos, direccion
  • Correo electronico
  • Numero de telefono
  • Fecha de nacimiento
  • Ubicacion geografica
  • Direccion IP

Datos de Uso:

  • Estadisticas de uso de la aplicacion
  • Telemetria o analisis de comportamiento
  • Horarios de uso
  • Frecuencia de uso

Datos Tecnicos:

  • Informacion del dispositivo
  • Sistema operativo
  • Especificaciones de hardware
  • Identificadores unicos del dispositivo
  • Logs de errores o diagnostico

Nota sobre identificadores de dispositivo:
Microsoft Store utiliza identificadores de dispositivo para gestionar licencias y sincronizar suscripciones entre tus dispositivos. Sin embargo, nosotros como desarrolladores NO tenemos acceso a estos identificadores. Microsoft Store nos proporciona unicamente una respuesta de "licencia valida" o "licencia invalida" sin revelar informacion del dispositivo.

Datos de Tareas:

  • Tus tareas programadas
  • Horarios de ejecucion
  • Nombres de tareas

TODO permanece en tu equipo local.

4. Almacenamiento de Datos

Datos Locales Unicamente

Toda la informacion que configures en Ninja Timer Control se almacena exclusivamente en tu dispositivo local en: %APPDATA%\

Archivos que se crean localmente:

  • Configuraciones de tareas
  • Preferencias de usuario
  • Idioma seleccionado
  • Tareas programadas

Estos archivos:

  • Solo existen en tu computadora
  • NO se sincronizan con ningun servidor
  • NO son accesibles para el desarrollador
  • Puedes eliminarlos en cualquier momento

5. Comunicaciones con Servidores Externos

Sin Conexion a Internet Requerida

Ninja Timer Control NO requiere conexion a internet para funcionar y NO se comunica con ningun servidor externo.

Excepciones:

1. Microsoft Store:

  • Cuando descargas o actualizas la aplicacion desde Microsoft Store
  • Para validar tu suscripcion o periodo de prueba
  • Gestionado Microsoft

2. Actualizaciones (Opcional):

  • Solo si eliges buscar actualizaciones manualmente
  • NO se envia informacion personal, solo la version actual de la app

NO HAY:

  • Servicios de analisis de terceros
  • Servicios de publicidad
  • Rastreadores
  • Cookies

Si decides suscribirte a Ninja Timer Control:

  • Microsoft Store procesa todos los pagos
  • Microsoft Store almacena tu informacion de pago
  • Microsoft Store identifica tu dispositivo y Microsoft Account
  • Nosotros NO tenemos acceso a:
    • Tu informacion de pago

Consulta la Politica de Privacidad de Microsoft Store.

10. Seguridad de los Datos

Proteccion Local

Dado que todos tus datos estan almacenados localmente en tu dispositivo:

  • Tu eres responsable de la seguridad fisica de tu computadora
  • Windows gestiona los permisos de acceso a archivos
  • Recomendamos:
    • Usar contrasena en tu cuenta de Windows
    • Mantener Windows actualizado
    • Usar antivirus confiable

Por nuestra parte:

  • NO almacenamos datos en servidores
  • NO hay riesgo de filtracion de datos de nuestro lado
  • NO hay brechas de seguridad hacia servidores
