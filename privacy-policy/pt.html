2. Compromisso com a Privacidade

Princípio Fundamental:

Ninja Timer Control NÃO coleta, NÃO armazena, NÃO transmite e NÃO compartilha quaisquer dados pessoais.

3. Dados que NÃO Coletamos

Este aplicativo NÃO coleta os seguintes tipos de dados:

Dados Pessoais:

  • Nome, sobrenome, endereço
  • E-mail
  • Telefone
  • Data de nascimento
  • Localização geográfica
  • Endereço IP

Dados de Uso:

  • Estatísticas de uso do aplicativo
  • Telemetria ou análise comportamental
  • Horários de uso
  • Frequência de utilização

Dados Técnicos:

  • Informações do dispositivo
  • Detalhes do sistema operacional
  • Especificações de hardware
  • Identificadores únicos de dispositivo
  • Logs de erro ou diagnóstico

Nota sobre identificadores de dispositivo:
A Microsoft Store utiliza identificadores para gerenciar licenças e sincronizar assinaturas entre dispositivos. Entretanto, nós como desenvolvedores NÃO temos acesso a esses identificadores. Recebemos apenas a resposta "licença válida" ou "licença inválida".

Dados de Tarefas:

  • Suas tarefas agendadas
  • Horários de execução
  • Nomes de tarefas

Tudo permanece na sua máquina local.

4. Armazenamento de Dados

Dados Somente Locais

Todas as informações configuradas no Ninja Timer Control são armazenadas exclusivamente no seu dispositivo em: %APPDATA%\

Arquivos criados localmente incluem:

  • Configurações de tarefas
  • Preferências do usuário
  • Idioma selecionado
  • Tarefas agendadas

Esses arquivos:

  • Existem apenas no seu computador
  • NÃO são sincronizados com nenhum servidor
  • NÃO são acessíveis ao desenvolvedor
  • Podem ser excluídos a qualquer momento
