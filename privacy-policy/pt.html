2. Compromisso com a Privacidade

Principio Fundamental:

Ninja Timer Control NAO coleta, NAO armazena, NAO transmite e NAO compartilha quaisquer dados pessoais.

3. Dados que NAO Coletamos

Este aplicativo NAO coleta os seguintes tipos de dados:

Dados Pessoais:

  • Nome, sobrenome, endereco
  • E-mail
  • Telefone
  • Data de nascimento
  • Localizacao geografica
  • Endereco IP

Dados de Uso:

  • Estatisticas de uso do aplicativo
  • Telemetria ou analise comportamental
  • Horarios de uso
  • Frequencia de utilizacao

Dados Tecnicos:

  • Informacoes do dispositivo
  • Detalhes do sistema operacional
  • Especificacoes de hardware
  • Identificadores unicos de dispositivo
  • Logs de erro ou diagnostico

Nota sobre identificadores de dispositivo:
A Microsoft Store utiliza identificadores para gerenciar licencas e sincronizar assinaturas entre dispositivos. Entretanto, nos como desenvolvedores NAO temos acesso a esses identificadores. Recebemos apenas a resposta "licenca valida" ou "licenca invalida".

Dados de Tarefas:

  • Suas tarefas agendadas
  • Horarios de execucao
  • Nomes de tarefas

Tudo permanece na sua maquina local.

4. Armazenamento de Dados

Dados Somente Locais

Todas as informacoes configuradas no Ninja Timer Control sao armazenadas exclusivamente no seu dispositivo em: %APPDATA%\

Arquivos criados localmente incluem:

  • Configuracoes de tarefas
  • Preferencias do usuario
  • Idioma selecionado
  • Tarefas agendadas

Esses arquivos:

  • Existem apenas no seu computador
  • NAO sao sincronizados com nenhum servidor
  • NAO sao acessiveis ao desenvolvedor
  • Podem ser excluidos a qualquer momento

5. Comunicacao com Servidores Externos

Sem Conexao Obrigatoria a Internet

Ninja Timer Control NAO requer conexao com a internet para funcionar e NAO se comunica com servidores externos.

Excecoes:

1. Microsoft Store:

  • Ao baixar ou atualizar pela Microsoft Store
  • Para validar sua assinatura ou periodo de teste
  • Gerenciado pela Microsoft

2. Atualizacoes (Opcional):

  • Somente se voce verificar atualizacoes manualmente
  • NAO sao enviados dados pessoais, apenas a versao do aplicativo

NAO ha:

  • Servicos de analise de terceiros
  • Redes de publicidade
  • Servicos de rastreamento
  • Cookies
